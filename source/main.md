# **1. Math operations**

| Operation | Description | Example | Result |
| --------- | ----------- | ------- | ------ |
| + | ... | 1 + 2 | 3 |
| - | ... | 3 - 2 | 1 |
| * | ... | 2 * 3 | 6 |
| / | ... | 3 / 2 | 1.5 |
| ^ | ... | 3 ^ 2 | 9 |
| %/% | ... | 9 %/% 4 | 2 |
| %% | ... | 9 %% 4 | 1 |
| & or && | ... | (3 > 2) & (2 < 1) | FALSE
| \| or \|\| | ... | (3 > 2) \| ( 2 < 1) | TRUE |
| ! | ... | !(3>2) | FALSE |
