# **Math operations**

| Operation | Description | Example | Result |
| --------- | ----------- | ------- | ------ |
| + | Addition | `1 + 2` | `[1] 3` |
| - | Subtraction | `3 - 2` | `[1] 1` |
| * | Multiplication | `2 * 3` | `[1] 6` |
| / | Division | `3 / 2` | `[1] 1.5` |
| ^ | Exponent | `3 ^ 2` | `[1] 9` |
| %/% | Integer division | `9 %/% 4` | `[1] 2` |
| %% | Modulus | `9 %% 4` | `[1] 1` |
| & or && | AND | `(3 > 2) & (2 < 1)` | `[1] FALSE`
| \| or \|\| | OR | `(3 > 2) \| ( 2 < 1)` | `[1] TRUE` |
| ! | NOT | `!(3>2)` | `[1] FALSE` |
